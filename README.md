# tiptap-sample

## 概要

- TipTapの実装サンプルです

## 動作環境

- Node.js - 18.x
- yarn - 1.22.x

## 使用ライブラリ

- vue - 3.2.x
- @tiptap/vue-3 - ^2.0.0-beta.199
- vuetify - 3.0.0-beta.11

## 動作確認

```bash
# パッケージインストール
$ yarn

# デバッグ実行
$ yarn dev
```
